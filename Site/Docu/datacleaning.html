datacleaning¶

Documentation des fonctions qui ont servi a faire du preprocessing sur les ImageGeoGuessing

Fonctions d’Histogrammes¶

DisplayFunctions.imageHist(image)¶

  • image : np.array

Génère et affiche l’histogramme d’une image, qu’elle soit en nuance de gris ou en couleurs

Est appelé par : showGrey

DisplayFunctions.showGrey(path, boolean)¶

  • path : String, contient le chemin d’accès vers une image

  • boolean : Bool, permet d’afficher ou non l’histogramme de l’image filtrée

Detection de murs¶

FiltragePhoto.detectWall(image, seuil, eps)¶

  • image : np.ndarray

  • seuil : float

  • eps : float

renvoie un booleen valant True si l’image est un mur, selon un critère basé sur la prédominance d’une couleur sur les autres

FiltragePhoto.detectVar(image, seuil)¶

  • image : np.ndarray

  • seuil : float

renvoie un booleen valant True si l’image est un mur, selon un critère basé sur la variance de l’image

FiltragePhoto.booleanHist(L)¶

  • L : Liste,

L est de la forme [[float0,boolean0],[float1,boolean1], …]

Permet de préparer des données pour faire un histogramme statistiques Utilisable pour des étudier les statistiques

renvoie le couple de triplet [varMaxTrue,varMoyTrue/nbr,varMinTrue],[varMaxFalse,varMoyFalse/nbr,varMinFalse (notation : varMaxTrue = variance maximale tel qu’un mur est détecté)

Fonctions de ProduitConvolution¶

ProduitConvolution.convuLocal(portionImage, filtre)¶

  • portionImage : np.ndarray

  • filtre : np.ndarray

  • Génère ValueError si les matrices ne sont pas de même taille

Renvoie le produit le convolution entre les deux matrices

ProduitConvolution.convuProduct(image)¶

  • image : np.ndarray de dimension n,p

renvoie le produit de convolution de image par le filtre f de dim(3x3)

f = -1, 0, 1,

-1, 0, 1, -1, 0, 1,
