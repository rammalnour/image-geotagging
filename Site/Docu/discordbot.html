Communication avec l’ia¶

DiscordBot39.getloc(num)¶

num : int num doit être compris entre 0 et 99

renvoie l’indice de la région associé au numéro num

DiscordBot39.get_REG_IA(path)¶

  • path : String

path contient le chemin d’accès a l’image qu’on soumet a l’IA

renvoie la région correspondant au choix de l’IA

headless_browser_vm.GetRepIA(path)¶

  • path : String

Crée un browser en mode headless ( sans interface graphique) charge la page et upload l’image située au chemin path renvoie la liste contenant les probas de chaque région

Pour lancer une partie de GeoGuessing, tapez ??play, et patientez que l’image ainsi que l’ia chargent a l’etape 0, une image est chosie aleatoirement parmi 100, et l’ia fait sa prediction pour tenter de deviner la region de l’image, tapez ??guess suivi du nom de la region Si la reponse est correcte, le score du joueur qui a bien repondu est augmente de 1
